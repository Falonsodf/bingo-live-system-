
# Bingo Virtual - Proyecto Completo

Incluye:
- Código React (Host + Player) con Firebase Realtime DB (placeholder config).
- Exportar cartones a PDF.
- Generación y verificación de bingo.
- Overlays para OBS / TikTok Live Studio.
- Guion de 1 minuto para promoción en TikTok.
- Instrucciones para generar APK con Capacitor.
- Guía de integración de pagos: PayPal + opciones Yape/Plin (referencias).

## Instalación rápida
1. Clona el repo o descomprime el ZIP.
2. Rellena `src/App.jsx` firebaseConfig con tus credenciales de Firebase.
3. `npm install`
4. `npm run start` (ejecuta Vite)

## Archivos importantes
- `src/App.jsx`: lógica principal (host/player).
- `public/overlays/`: plantillas HTML/CSS para overlays.
- `README_PAYMENT.md`: guía de integraciones de pago.
- `CAPACITOR.md`: pasos para crear APK usando Capacitor.
